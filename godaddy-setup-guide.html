PART B

üßë‚Äçüíº Your Steps in GoDaddy

B1

Enter your email address and password. If you have trouble signing in, click "Forgot Password" ‚Äî GoDaddy will send a reset link to your email.

B2

Navigate to the DNS settings for denmanpaddling.ca

After signing in, follow these clicks exactly:

Click path through GoDaddy menus:

B3

Delete the existing "A" record (if there is one)

Removes the old record so we can add new ones

Look at the table. If you see a row with Type = A and Name = @, you need to delete it first.

Click the pencil icon ‚úèÔ∏è or the three dots next to it ‚Üí choose Delete.

Don't worry: If you don't see any "A" record with "@", just skip to Step B4.
Important: Only delete records with Type = A and Name = @. Do not delete any other records (especially MX records ‚Äî those are for email).
B4

Add the 4 new "A" records

These point your domain at GitHub's servers

GitHub's website files are stored on servers around the world. These four IP addresses are the addresses of those servers ‚Äî we add four so that if one goes down, the others keep your website running. Think of it like giving someone four phone numbers for the same office.

Each record simply tells GoDaddy: "When someone types denmanpaddling.ca into their browser, send them to this address."

Click the "Add" or "Add New Record" button and add the four records one at a time:

Type Name Value (copy exactly) TTL
A @ 185.199.108.153 1 Hour
A @ 185.199.109.153 1 Hour
A @ 185.199.110.153 1 Hour
A @ 185.199.111.153 1 Hour

For each record: select Type = A, Name = @, paste the IP address, leave TTL as "1 Hour", click Save. Then click Add again for the next one.

B5

Add the "CNAME" record for www

Makes www.denmanpaddling.ca work too

Click Add again, this time for a different type of record:

Type Name Value (use your GitHub username) TTL
CNAME www [your GitHub username].github.io 1 Hour
Use your GitHub username ‚Äî the name you chose when creating your account. It looks like: denmanpaddling.github.io (with your actual username)
B6

Save everything and wait

Changes take up to 48 hours to spread across the internet

After adding all 5 records, click Save one final time. You're done in GoDaddy ‚Äî you can close the tab.

Normal waiting time: Usually 1‚Äì4 hours. Maximum 48 hours. During this time, the website may not show up yet ‚Äî that's completely normal.

To check if it's working: open a new browser tab, type denmanpaddling.ca and press Enter. When you see your new website, everything is done!
